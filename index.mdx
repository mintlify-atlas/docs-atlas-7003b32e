## Overview

SMyEG is a comprehensive geospatial monitoring and evaluation platform designed for forest patrimony management. Built with Next.js, PostgreSQL, and modern mapping technologies, it provides organizations with powerful tools to manage forest resources, track biological assets, and analyze land use patterns.

## Technology Stack

SMyEG is built with modern, production-ready technologies:

- **Frontend**: Next.js 16 (App Router) with React 19
- **Backend**: Node.js with Next.js API Routes
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth v5 with credential-based authentication
- **Mapping**: Leaflet and React Leaflet for geospatial visualization
- **Background Jobs**: Custom worker schedulers for asynchronous processing
- **UI Components**: Tailwind CSS with shadcn/ui

## Getting Started

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install required packages
    
    ```bash
    git clone https://github.com/drosales10/SMyEG.git
    cd SMyEG
    npm install
    ```
  </Step>
  
  <Step title="Configure Environment">
    Set up your environment variables and database connection
    
    ```bash
    cp .env.example .env
    docker compose up -d
    ```
  </Step>
  
  <Step title="Initialize Database">
    Run Prisma migrations and seed initial data
    
    ```bash
    npm run db:generate
    npm run db:migrate
    npm run db:seed
    ```
  </Step>
  
  <Step title="Start Development Server">
    Launch the application in development mode
    
    ```bash
    npm run dev
    ```
  </Step>
</Steps>

<Note>
The default admin credentials are `admin@example.com` / `Admin1234`. Change these immediately in production environments.
</Note>
